🔍 Need Help Finding The Path?

This tool scans domains for hidden endpoints, WordPress vulnerabilities, and exposed directories.

📥 Download PathTraversalBot

Python script to scan any domain for hidden paths and endpoints.

⬇ Download bot.py ⬇ Download install.bat

📋 Requirements

Run install.bat first to install all dependencies.

pip install requests

🛠 How to use

1. Run install.bat to install dependencies
2. Run pathtraversalbot.py
3. Enter domain(s) to scan (separated by comma)
4. Review the generated report

📂 What it finds

- Hidden admin panels
- WordPress endpoints
- Exposed APIs
- Backup files
- Configuration files
- Path traversal vulnerabilities

Use responsibly. Only scan domains you own or have permission to test.
